## 스터디 방식 

1. 푼 문제는 github에서 공유

2. 교과서를 기준으로 각자 문제를 풀어보고 문제 리뷰 준비(발표는 랜덤프로그램으로 지정)
   * 발표 형식은 자유

3. 준비가 안되어있을시 또는 결석 시 벌금 (* 벌금은 오티 때 다같이 정하기)

4. 잦은 결석은 퇴출

5. 방학때는 온라인으로 코드 업로드하는 방식으로 진행될 예정. (오프라인 미팅 없음)
